Add IMDb syntax functionallity in your post. Enter simple tags and this plugin
replace with IMBb data direct from IMBb Database.

== Description ==
This plugin extends your writing with an markup syntax for display movie data in your
post content and title. Datasource is native IMDb Web API same datasource IMDb:s
mobile apps using.
